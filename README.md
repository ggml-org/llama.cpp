# Lemonade llama.cpp fork

This repo is a "Github actions only" fork of llama.cpp that is built with more ROCm releases than upstream llama.cpp supports.

These are built for the purpose of lemonade use, but they can be utilized by any project.  Releases just need to be used with matching AMD ROCm SDK releases.

# Cadence

The builds are triggered nightly, or can optionally be triggered on demand as well.
