## Project Selection Guide

### Choose Project 1 (RAG) if you want to learn:
- Document processing and management
- Vector databases and similarity search
- Full-stack web development
- Microservices architecture

### Choose Project 2 (Multi-Platform Chat) if you want to learn:
- Cross-platform mobile development
- Real-time communication
- State management
- Offline-first architecture

### Choose Project 3 (Agent) if you want to learn:
- Function calling and tool use
- Multi-step reasoning
- Agent architectures
- Complex workflow orchestration

### Choose Project 4 (Mobile Inference) if you want to learn:
- Mobile deployment
- Resource optimization
- Native development
- Performance tuning

## General Requirements

All projects must include:

1. **Source Code**
   - Clean, well-documented code
   - Modular architecture
   - Error handling
   - Unit tests (minimum 60% coverage)

2. **Documentation**
   - README with setup instructions
   - Architecture diagram
   - API documentation (if applicable)
   - User guide

3. **Deployment**
   - Docker Compose setup OR
   - Cloud deployment instructions OR
   - App store build process

4. **Demo**
   - 5-10 minute video walkthrough
   - Screenshots of key features
   - Sample use cases

## Evaluation Criteria

All projects evaluated on:

- **Functionality** (30%): Core features working correctly
- **Code Quality** (25%): Clean, maintainable, tested code
- **Documentation** (20%): Complete and clear documentation
- **UX/Design** (15%): Intuitive and polished interface
- **Innovation** (10%): Creative features or optimizations

## Bonus Points

Earn up to +20% extra credit for:

- Advanced features beyond requirements
- Outstanding code quality and testing
- Production deployment (live demo)
- Open source contribution
- Technical blog post about the project

## Submission

Submit via:
1. GitHub repository (public or private)
2. Project report (PDF)
3. Demo video (unlisted YouTube link)

Include:
- Source code with README
- Setup and deployment instructions
- Test results
- Performance benchmarks (if applicable)
- Reflection on challenges and learning
